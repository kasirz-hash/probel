Пробелы для Instagram

Вставьте текст — приложение заменит пустые строки на невидимый пробел. Instagram сохранит отступы между абзацами.
Исходный текст
0 символов
Символ для пустых строк
Готовый текст
0 символов
Важно: пустые строки будут выглядеть пустыми, но внутри будет невидимый символ. Поэтому Instagram обычно не удаляет такие отступы при вставке текста.
